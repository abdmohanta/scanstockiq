# scanstockiq
A user-friendly Inventory Management System using Spring Boot MySQL that supports UPC barcode scanning. When a code is scanned, the system will auto identify the product n update inventory levels in real time. Application will include product management, stock tracking, inventory history, and reporting features, with a simple inventory operations.
